# What is haunt?

Haunt is a biolink platfom permitting to express yourself and introduce you on your own page, with tons of customization options.

- Register : https://haunt.gg/register
- Login : https://haunt.gg/login

You can check any profile with the haunt command to see information about someone's haunt page:

<CodeGroup>
    ```javascript Syntax
    ,haunt (username)
    ```
     ```javascript Example
    ,haunt vani
    ```
</CodeGroup>
