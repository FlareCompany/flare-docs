## What is vanity link ?

Vanity link automatically tracks users that puts your vanity link in your server and give them a role (more rewards incoming soon).

This only affects users who have your vanity in their **custom status**.

<Warning>
Your server must be **boosted at level 3** (14 boosts)
</Warning>

## Setting up vanity link

### Adding the link

Use the `vanity add /vanity` to set you vanity on your server.

### Setting up the channel

Select in what channel your message will be sent, check here for the message setup.
Use the `vanity channel set  #channel` with your channel.

You can also **remove** the channel by running the `vanity channel remove #channel`.

### Setting up the role 

You can add roles to users putting your custom vanity link in their status.

Use the `vanity role add @role` command to add a role that the bot will add to whenever reps the vanity.

Multiple roles can be added. You can remove the roles by using the `vanity role remove @role`.

### Setting up the message

You can create your own message whenever someone reps the vanity, in your selected channel.
Embed code can be used in the message. You can setup the message with the `vanity message $v{...}` command.

### Removing vanity link

You can simply remove the vanity configuration by running the `vanity remove`.

<Note>
    All the data will stay so if your enable the vanity again: the message, role(s), the channel and the vanity will be saved.
</Note>
